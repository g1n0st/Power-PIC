 # Power-PIC
Taichi Implementation of "The Power Particle-in-Cell Method"

Running the Demo
----------------

```bash
python3 main_2d.py
```

parameters:

```bash
usage: main_2d.py [-h] [--flip] [--save] [--res RES] [--t_res T_RES] [--show SHOW]

optional arguments:
  -h, --help     show this help message and exit
  --flip
  --save
  --res RES
  --t_res T_RES
  --show SHOW
```
